# 8-Bit Highway

A retro-style 2D car racing game made with Processing.

## Features

- Player car movement
- Moving road and lane lines
- Enemy cars
- Collision detection
- Score system
- Game over screen
- Restart option
- Increasing speed over time

## How to Run

1. Install Processing 4.
2. Open `EightBitHighway/EightBitHighway.pde`.
3. Click the Run button.
